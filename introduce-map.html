오시는 길

찾아오시는 길

  • 주소 : 경상북도 문경시 문경읍 새재1길 45(하초리 243)

  • 전화 : 054-550-8383

  • 팩스 : 054-550-6179

자가용으로 오실 때

  • 서울 : 경부고속도로 → 호법분기점 → 영동고속도로 → 여주분기점 → 중부내륙고속도로 → 문경새재IC (문경새재도립공원방향) → 문경새재 자연생태전시관/생태공원 (1시간 40분 소요)

  • 부산, 대구 : 경부고속도로 → 김천분기점 → 중부내륙고속도로 → 문경새재IC(문경새재도립공원방향) → 문경새재자연생태전시관/생태공원 (대구: 1시간 20분 소요)

  • 대전 : 경부고속도로 → 청원분기점 → 청원-상주간고속도로 → 낙동분기점 → 중부내륙고속도로 → 문경새재IC (문경새재도립공원방향)→ 문경새재자연생태전시관/생태공원 (2시간 소요)

  • 광주 : 88고속도로 → 경부고속도로 → 김천분기점 → 중부내륙고속도로 → 문경새재IC (문경새재도립공원방향) → 문경새재자연생태전시관 /생태공원 (4시간 소요)

버스로 오실때 (고속버스)

  • 서울 : 동서울터미널 → 문경읍(점촌) 2시간 소용 (배차간격30분)

  • 대전 : 동부시외버스터미널 → 문경읍(점촌) 3시간소요(배차간격30분)

  • 대구 : 북부터미널 → 문경읍(점촌) 2시간소요(배차간격15분)

기차로 오실때 (철도)

  • 서울방면 : 서울역 → 김천역 → 점촌역(영주, 강릉방면)

  • 대전방면 : 대전역 → 김천역 → 점촌역(영주, 강릉방면)

  • 부산방면 : 대구역 → 김천역 → 점촌역(영주, 강릉방면)

  • 강릉방면 : 영주역 → 안동역 → 점촌역(부산방면)
